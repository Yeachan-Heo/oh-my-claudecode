# 포스트 작성 가이드 (Writing Guide Ops)

> 이 파일은 `src/agents/post-writing-guide.md`에서 분리된 **작성 가이드**다.
> 조회수 1만+ 포스트 209개 분석 기반 (2026-03-20)

---

## 5대 패턴

### 패턴 A: "댓글로 알려줘" (참여 유도형) — 최고 134만뷰

```
협찬제품 샀다가 별로라서 후회한적있어? 이건 절대 사지마라 하는거 댓글로 적어줘.
```
- 첫 문장에서 질문 던짐
- "댓글로 적어줘/알려줘" 직접 요청
- 본인 경험 1개를 먼저 깔아서 댓글 장벽 낮춤

### 패턴 B: "이거 써봤는데 인생템" (솔직 후기형) — 최고 20만뷰

```
더마팩토리 나이아신아마이드 20%세럼 모르는 사람 없게 해주세요.
얼굴 새 하얘짐. N차 공병템. 80ml에 1만600원.
```
- 제품명 직접 언급 (숨기지 않음)
- 가격 명시 → 가성비 어필
- "광고 아님" 한마디로 신뢰 확보

### 패턴 C: "리스트형 정보" (번호 나열형) — 최고 61만뷰

```
최근 5년간 했던 소비 중 가장 후회 없는 것
1. 다이슨 드라이기
2. 애플 맥북
3. 에어프라이어
```
- 숫자로 구조화 (스크롤하면서 훑기 좋음)
- 각 항목에 "전(문제) → 후(해결)" 구조
- 제목이 곧 훅

### 패턴 D: "반전/놀람" (의외성형) — 최고 56만뷰

```
피부에 진심인 의사로서 말합니다.
실내에선 굳이 선크림 바르지 마시구요.
```
- 권위자 포지션 ("의사로서", "약사가", "10년 차")
- 상식을 뒤집는 첫 문장
- 짧고 단정적인 문장

### 패턴 E: "실패/비추" (경고형) — 최고 14만뷰

```
왜 유명한지 모르겠는 비추템 3가지
1. 에스네이처 스쿠알란 — 밀리고요.. 밀리네요;;
2. 듀이트리 선크림 — 냄새 웩 🤢
```
- "비추"가 "추천"보다 신뢰도 높음
- 감정 표현 진솔함
- 구체적 문제점 명시

## 훅 공식

| 유형 | 구조 | 조회수 상위 예시 |
|------|------|----------------|
| **질문형** | "~한 사람 나만 그래?" | "보조배터리 못 들고 탄다고 잡힘" (110만) |
| **단정형** | "~임. 끝." | "무인양품 가면 꼭 사올 것 = 이 사진" (97만) |
| **반전형** | "~라고? 아닌데" | "선크림 바르지 마세요" (56만) |
| **팩트형** | "~가 OO인데" | "회사 언니가 목이버섯 먹는다길래 뭔데?" (1.5만) |
| **경고형** | "~하지 마" | "이건 절대 사지마라" (134만) |

**훅 핵심 규칙**:
- **20자 이내** (30자도 길다)
- 첫 줄에 팩트/숫자/반전 1개 필수
- 제품명은 첫 줄에 넣지 않음 (광고 냄새)
- "~합니다", "~드립니다" 절대 금지

## 글쓰기 공식

### 최우선 규칙: 정보 밀도

> **공감만 있는 포스트 = 죽는다.** 반드시 구체적 정보를 넣어야 한다.

**정보 밀도 필수 요소 (최소 1개)**:
- 구체적 숫자: "10배", "5분", "1만600원"
- 구체적 제품/브랜드명: "더마팩토리", "다이슨"
- 구체적 해결법: 레시피, 사용법, 순서, 비교표
- 구체적 리스트: "1. OO 2. OO 3. OO"

### 글자수 기준

| 구간 | 평균 글자수 | 포스트 수 |
|------|-----------|----------|
| **10K+ (대박)** | **117자** | 209개 |
| 5K-10K | 131자 | 68개 |
| 1K-5K | 126자 | 261개 |
| 0-100 | 138자 | 151개 |

**길수록 조회수가 떨어진다.** 목표: 100~150자. 최대 200자. 500자 절대 금지.

### 문체 규칙

**DO**:
- 구어체 ("걍", "ㅋㅋ", "ㅜ", "ㄹㅇ")
- 짧은 문장 (한 줄에 하나의 정보)
- 줄바꿈 많이 (한 문단 1~2줄)
- 감정 솔직하게 ("빡쳤다", "신세계")
- "광고 아님" 한마디
- 가격 직접 명시 ("1만600원")
- 구체적 숫자 ("시금치 10배", "5분컷")

**DON'T**:
- 공감만 있고 정보 없는 글
- 채널 카테고리 이탈
- 200자 초과
- 문어체 ("~하는 것이 좋습니다")
- 이모지 과다 (3개 초과)
- "여러분", "추천드립니다", "꼭 사세요"
- 해시태그 5개 이상
- 모호한 CTA ("댓글 좀", "힘이 됨✨")

## 카테고리별 조회수 패턴

| 카테고리 | 잘 되는 구조 |
|---------|------------|
| **뷰티** | 비추+솔직후기, "이거 써봐" |
| **생활** | 리스트형, "이거 사" |
| **건강** | 전문가형, 반전형 |
| **다이어트** | 실패담, 현실적 식단 |

## 조회수 터지는 소재 7유형

| 순위 | 소재 | 왜 터지는가 |
|------|------|-----------|
| 1 | **비추/실패담** | 경고 = 신뢰, 공유 욕구 |
| 2 | **가성비 발견** | 가격 대비 충격 |
| 3 | **생활 꿀팁** | 즉시 실행 가능 |
| 4 | **리스트 정리** | 저장+공유 유도 |
| 5 | **전문가 반전** | 권위+의외성 |
| 6 | **공감 질문** | 소속감 |
| 7 | **Before/After** | 변화 충격 |
