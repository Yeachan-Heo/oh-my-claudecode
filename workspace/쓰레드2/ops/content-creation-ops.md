## 운영 모드

### 워밍업 모드 (published_count < 100)
- 셀프댓글 생략: `self_comments` 빈 배열 `[]` 반환
- 광고 요소 제거: 제휴링크 표시, 프로필 유도, CTA 전부 제거
- 순수 콘텐츠만: 니즈/공감 기반 + **구체적 정보 필수**
- 목적: 계정 신뢰도 구축, 알고리즘 워밍업

### 일반 모드 (published_count >= 100)
- 셀프댓글 2개 생성
- 제휴링크 표시 포함
- 프로필 유도 포함

## Chain-of-Thought 6단계 실행 절차

**반드시 아래 6단계를 순서대로 밟는다. 단계를 건너뛰지 않는다.**

### Step 1: 포지셔닝 카드 이해

`data/briefs/{today}_positioning.json` Read.

각 상품의 포지셔닝 카드를 하나씩 읽고 정리:
- 이 상품의 **니즈**(need_id)는 뭔가?
- 어떤 **포맷**으로 접근하는가?
- **angle**과 **tone**의 핵심 메시지는 뭔가?
- **avoid** 리스트에 뭐가 있나?

### Step 2: 학습 피드백 반영

`data/learnings/latest.json` Read (없으면 스킵).

| 필드 | 반영 방법 |
|------|----------|
| `winning_hooks` | 잘 먹힌 훅 타입에 가중치 |
| `winning_formats` | 성과 높은 포맷 더 정교하게 |
| `losing_patterns` | 해당 패턴 본문에서 회피 |
| `winning_categories` | 해당 카테고리 공감 포인트 녹이기 |

### Step 3: 본문 3개 작성 (서로 다른 전략)

| Draft | 전략 | 시작 방식 |
|-------|------|----------|
| Draft 1: 공감형 | 문제/불편함 공감 | "~하는 사람 나만 아니지?" |
| Draft 2: 스토리형 | 1인칭 경험담 | "지난달에 ~했는데" |
| Draft 3: 호기심형 | 의외의 발견/반전 | "~인 줄 알았는데" |

**다양성 체크**: 3개 본문의 첫 문장이 같은 구조면 실패.

### Step 4: 대안 훅 5개 생성

| 훅 타입 | 패턴 |
|---------|------|
| 질문형 | "~ 나만 그런 거 아니지?" |
| 반전형 | "~ 인 줄 알았는데" |
| 공감형 | "~ 하는 사람 손" |
| 숫자형 | "N개 써봤는데" |
| 고백형 | "솔직히 말하면" |

**훅 품질 규칙**:
- 제품명 직접 노출 절대 금지
- **20자 이내** (30자도 길다)
- 니즈/공감/호기심 중심

### Step 5: 셀프댓글 + 프로필 유도

**워밍업 모드**: 이 단계 완전히 건너뜀

**일반 모드**:
- 댓글 1: 대화 유도형
- 댓글 2: 정보 보충형 (제휴링크 자리 포함)
- 프로필 유도: "자세한 건 프로필에 정리해놨음" 형태

**셀프댓글 금지 패턴**:
- "좋은 정보 감사합니다", "잘 읽었습니다", "유익한 정보"
- "링크 드릴게요" 식 직접 유도

### Step 6: Self-Verification (자기 검증)

#### 킬러 게이트 (실패 시 Step 3부터 재작성)

| # | 체크 |
|---|------|
| K1 | 구체적 정보 있는가? (숫자/제품명/팁/리스트 중 1개) |
| K2 | 이미지 지정되어 있는가? |
| K3 | 뷰티/건강/생활/다이어트 범위 내인가? |
| K4 | 200자 이내인가? (목표 100~150자) |

#### AI 말투 탐지 체크리스트

| # | 금지 패턴 |
|---|----------|
| 1 | "~하는 것이 중요합니다" |
| 2 | "여러분" |
| 3 | "추천드립니다" |
| 4 | "효과적입니다" |
| 5 | 3줄 이상 연속 문어체 |
| 6 | 이모지 3개 초과 |
| 7 | "최고의", "완벽한", "놀라운" |
| 8 | "강력 추천", "꼭 사세요" |

## 포맷 다양화 규칙

6가지 포맷을 라운드로빈 + 니즈 성격 기반으로 순환. **같은 포맷이 연속 3회 이상 나오면 안 된다.**

| 포맷 | 적합한 니즈 | 구조 |
|------|-----------|------|
| 문제공감형 | 불편해소, 외모건강 | 공감 → 발견 → 솔직 소감 |
| 솔직후기형 | 성과향상, 외모건강 | 경험 → 과정 → 결과 |
| 비교형 | 시간절약, 돈절약 | N개 써봤는데 → 차이점 → 결론 |
| 입문추천형 | 자기표현, 시간절약 | 초보 시절 → 시행착오 → 지금 쓰는 거 |
| 실수방지형 | 불편해소 | 실수 경험 → 교훈 → 대안 |
| 비추천형 | 돈절약, 외모건강 | 기대 → 실망 → 차라리 이거 |

## 법적 Guardrail

### 공정거래위원회 「추천/보증 심사 지침」

경제적 이해관계(제휴링크 수수료)가 있으면 **반드시 표시**.

표시 방법 (택 1):
- 본문 마지막: "제휴링크 포함"
- 해시태그: `#광고` 또는 `#제휴링크포함`
- 셀프댓글: "이 글에는 제휴 링크가 포함되어 있음"

### 건강기능식품 특별 규칙
- "치료", "예방", "효능" 절대 금지
- 허용: "나한테는 맞았음", "좀 달라진 느낌"
- 금지: "~에 효과적", "~를 치료/예방"
