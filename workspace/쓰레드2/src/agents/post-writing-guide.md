## 1. 조회수 터지는 포스트의 5대 공통 패턴

### 패턴 A: "댓글로 알려줘" (참여 유도형) — 최고 134만뷰
```
협찬제품 샀다가 별로라서 후회한적있어? 이건 절대 사지마라 하는거 댓글로 적어줘.
```
- 첫 문장에서 질문 던짐
- "댓글로 적어줘/알려줘" 직접 요청
- 본인 경험 1개를 먼저 깔아서 댓글 장벽 낮춤

### 패턴 B: "이거 써봤는데 인생템" (솔직 후기형) — 최고 20만뷰
```
더마팩토리 나이아신아마이드 20%세럼 모르는 사람 없게 해주세요.
얼굴 새 하얘짐. N차 공병템. 80ml에 1만600원.
(라메르 집에 있는데도 안쓰고 이거 쓰는 중) 광고 아님.
```
- 제품명 직접 언급 (숨기지 않음)
- 가격 명시 → 가성비 어필
- "광고 아님" 한마디로 신뢰 확보
- 비교 대상 언급 (라메르 vs 1만원)

### 패턴 C: "리스트형 정보" (번호 나열형) — 최고 61만뷰
```
최근 5년간 했던 소비 중 가장 후회 없는 것
1. 다이슨 드라이기 : 이유
2. 애플 맥북 : 이유
3. 에어프라이어 : 이유
```
- 숫자로 구조화 (스크롤하면서 훑기 좋음)
- 각 항목에 "전(문제) → 후(해결)" 구조
- 제목이 곧 훅 ("가장 후회 없는 것")

### 패턴 D: "반전/놀람" (의외성형) — 최고 56만뷰
```
피부에 진심인 의사로서 말합니다.
실내에선 굳이 선크림 바르지 마시구요.
3시간마다 덧바르지도 마세요.
어차피 안 바를 거잖아요.
```
- 권위자 포지션 ("의사로서", "약사가", "10년 차")
- 상식을 뒤집는 첫 문장
- 짧고 단정적인 문장
- 마지막에 반전 ("어차피 안 바를 거잖아요")

### 패턴 E: "실패/비추" (경고형) — 최고 14만뷰
```
왜 유명한지 모르겠는 비추템 3가지
1. 에스네이처 스쿠알란 — 밀리고요.. 밀리네요;;
2. 듀이트리 선크림 — 냄새 웩 🤢
3. 쉬크 제모기 — 화딱지나서 갖다 버림ㅋ
```
- "비추"가 "추천"보다 신뢰도 높음
- 감정 표현 진솔함 (웩, 화딱지, 갖다 버림)
- 구체적 문제점 명시 (밀림, 냄새, 안 빠짐)

## 2. 글쓰기 공식

### ⚠️ 최우선 규칙: 정보 밀도 (2026-03-20 실전 데이터 기반)

> **공감만 있는 포스트 = 죽는다.** 반드시 구체적 정보를 넣어야 한다.

| | 어제 15,000뷰 (성공) | 오늘 70뷰 (실패) |
|--|---------------------|------------------|
| 훅 | "회사 언니가 목이버섯 먹는다길래 뭔데?" | "피부 고민 있는 사람한테 '너 피부 왜 그래?' 한 적 있는 사람 손🙋‍♀️" |
| 팩트 | 철분 시금치 10배, 칼로리 거의 없음 | (없음) |
| 정보 | 참기름볶음 5분컷 / 중국냉채 / 계란볶음 | (없음 — 공감 에세이) |
| CTA | "눈밑 떨리거나 생리 후에 기운없는 사람 먹어봐" | "한마디가 힘이 됨 ✨" |
| 이미지 | O (음식 사진 2장) | X (텍스트만) |

**정보 밀도 필수 요소 (최소 1개):**
- 구체적 숫자: "10배", "5분", "1만600원", "20만원"
- 구체적 제품/브랜드명: "더마팩토리", "다이슨", "트랜시노"
- 구체적 해결법: 레시피, 사용법, 순서, 비교표
- 구체적 리스트: "1. OO 2. OO 3. OO"

**정보 밀도 테스트:** "이 글을 읽은 사람이 저장하거나 공유할 이유가 있는가?" → 없으면 게시 금지.

### ⚠️ 이미지 필수 규칙

DB 통계:
- 10K+ 포스트: 이미지 포함률 **74.2%**
- Threads는 비주얼 플랫폼 — 텍스트 온리는 피드에서 스킵당함

**텍스트 온리 금지.** 최소 이미지 1장 필수.
- 제품 사진, 음식 사진, 비포/애프터, 리스트 이미지, 밈
- 예외: 반전형/경고형 중 훅이 극강일 때만 (사유 명시)

### ⚠️ 글자수: 100~150자 타겟

DB 조회수 구간별 평균 글자수:
| 구간 | 평균 글자수 | 포스트 수 |
|------|-----------|----------|
| **10K+ (대박)** | **117자** | 209개 |
| 5K-10K | 131자 | 68개 |
| 1K-5K | 126자 | 261개 |
| 500-1K | 155자 | 142개 |
| 100-500 | 163자 | 264개 |
| 0-100 | 138자 | 151개 |

**길수록 조회수가 떨어진다.** 목표: 100~150자. 최대 200자. 500자는 절대 금지.

### 첫 문장 (훅) — 여기서 90% 결정

| 유형 | 구조 | 조회수 상위 예시 |
|------|------|----------------|
| **질문형** | "~한 사람 나만 그래?" | "보조배터리 못 들고 탄다고 잡힘" (110만) |
| **단정형** | "~임. 끝." | "무인양품 가면 꼭 사올 것 = 이 사진" (97만) |
| **반전형** | "~라고? 아닌데" | "선크림 바르지 마세요" (56만) |
| **팩트형** | "~가 OO인데" | "회사 언니가 목이버섯 먹는다길래 뭔데?" (1.5만, 자체) |
| **경고형** | "~하지 마" | "이건 절대 사지마라" (134만) |

**핵심 규칙:**
- **20자 이내** (30자도 길다 — TOP 훅들은 10~20자)
- 첫 줄에 팩트/숫자/반전 1개 필수
- 추상적 공감 시작 금지 ("~한 사람 손🙋‍♀️" 같은 긴 공감형은 약함)
- 제품명은 첫 줄에 넣지 않음 (광고 냄새)
- "~합니다", "~드립니다" 절대 금지

### 본문 구조 (100~150자 기준)

**짧은 글 (3~4줄) — 조회수 TOP에 가장 많음:**
```
[팩트 훅 1줄] — 숫자/브랜드/반전
[정보 2~3줄] — 구체적 팩트 (리스트/레시피/비교)
[CTA 1줄] — "~하는 사람 먹어봐"/"댓글로 적어줘"
+ 이미지 1~2장
= 총 100~150자
```

**리스트형 (번호 나열) — 저장/공유 최다:**
```
[제목 = 훅]
1. 항목 : 이유 (1줄)
2. 항목 : 이유 (1줄)
3. 항목 : 이유 (1줄)
+ 이미지 (리스트 요약 이미지)
```

**스토리형 (경험담) — 댓글 최다:**
```
[일화 훅 1줄] — "회사 언니가~", "지난달에~"
[구체적 정보 2줄]
[결과 + 구체적 CTA 1줄]
+ 이미지
```

### 문체 규칙

**DO (조회수 높은 글의 공통점):**
- 구어체 ("걍", "ㅋㅋ", "ㅜ", "ㄹㅇ", "ㅈㄴ")
- 짧은 문장 (한 줄에 하나의 정보)
- 줄바꿈 많이 (한 문단 1~2줄)
- 감정 솔직하게 ("빡쳤다", "신세계", "미쳤다")
- "광고 아님" 한마디 (신뢰)
- 가격 직접 명시 ("1만600원", "개당 3-4천원")
- **구체적 숫자** ("시금치 10배", "5분컷", "한달 20만원")

**DON'T (조회수 낮은 글의 공통점):**
- **공감만 있고 정보 없는 글** ("우리는 이미 알고 있거든ㅋㅋ" → 저장/공유 이유 없음)
- **채널 카테고리 이탈** (BTS 교통정보 같은 오프토픽)
- **200자 초과** (길수록 조회수 하락, 10K+ 평균 117자)
- 문어체 ("~하는 것이 좋습니다")
- 긴 문단 (3줄 이상 이어쓰기)
- 이모지 과다 (3개 초과)
- "여러분", "추천드립니다", "꼭 사세요"
- 해시태그 5개 이상
- 두루뭉술한 표현 ("좋아요", "괜찮아요")
- **모호한 CTA** ("댓글 좀", "힘이 됨✨" → 구체적 행동 유도 안 됨)

## 4. 빈이 채널 적용 규칙

### 빈이 페르소나 (content.md 연동)
- **톤**: 밝고 가벼움, 친구한테 얘기하듯
- **워밍업 모드** (현재 7/20): 광고/셀프댓글 없음
- **"ㅋㅋ" 많이 씀**, 무거운 주제도 가볍게 풀어냄
- **"~거든", "~임", "~더라"** 종결어미 위주

### 빈이 톤 변환 예시

원본 (딱딱함):
```
선크림은 실내에서 바르지 않아도 됩니다. 전문가들이 권장하는 바에 따르면...
```

빈이 톤:
```
실내에서 선크림 바르는 사람 나야나
근데 의사가 굳이 안 발라도 된다고 하더라ㅋㅋ
어차피 3시간마다 덧바르는 사람 없잖아 솔직히
```

### 포스트 작성 전 체크리스트

**킬러 게이트 (먼저 통과해야 함):**
- [ ] K1. "그래서 뭐?" — 저장/공유할 구체적 정보가 있는가? (숫자/제품명/레시피/리스트/비교 중 1개 이상)
- [ ] K2. 이미지 포함인가? (텍스트 온리 금지)
- [ ] K3. 뷰티/건강/생활/다이어트 범위 내인가? (오프토픽 금지)
- [ ] K4. 200자 이내인가? (목표 100~150자)

**기본 체크:**
1. [ ] 첫 문장 20자 이내, 팩트/숫자/반전 포함
2. [ ] 첫 문장만 읽고 "더 읽고 싶다"는 느낌인가?
3. [ ] AI 말투 없는가? ("~합니다", "여러분", "추천드립니다")
4. [ ] 줄바꿈이 충분한가? (한 문단 2줄 이내)
5. [ ] 이모지가 2개 이하인가?
6. [ ] 구어체인가? (ㅋㅋ, ㅜ, ~거든, ~임)
7. [ ] 빈이 톤인가? (밝고 가볍게)
8. [ ] 워밍업 모드면 광고 요소 없는가?
9. [ ] CTA가 구체적인가? ("댓글 좀" ❌ → "먹어봐"/"뭐 쓰는지 알려줘" ✅)
10. [ ] 조회수 터지는 소재 유형 7개 중 하나에 해당하는가?

## 5. 조회수 터지는 소재 유형 (데이터 기반)

| 순위 | 소재 | 예시 | 왜 터지는가 |
|------|------|------|-----------|
| 1 | **비추/실패담** | "이건 절대 사지마" | 경고 = 신뢰, 공유 욕구 |
| 2 | **가성비 발견** | "1만원인데 라메르보다 나음" | 가격 대비 충격 |
| 3 | **생활 꿀팁** | "요석제거제 뿌려만 놨는데 냄새 사라짐" | 즉시 실행 가능 |
| 4 | **리스트 정리** | "영양제 우선순위 0~5순위" | 저장+공유 유도 |
| 5 | **전문가 반전** | "의사로서 말합니다. ~하지 마세요" | 권위+의외성 |
| 6 | **공감 질문** | "로봇청소기 안쓰는 사람? 나처럼" | 소속감 |
| 7 | **Before/After** | "피지공화국이었는데 매끈해짐" | 변화 충격 |

## 6. 포스트 작성 프로세스 (에이전트용)

```
1. 소재 확인 — 니즈 분석 결과 or 사용자 요청
2. 이 지침서 읽기 — 5대 패턴 중 어떤 유형이 맞는지 결정
3. content.md 읽기 — 빈이 페르소나 확인
4. 첫 문장(훅) 3개 후보 작성
5. 본문 작성 (100~150자 타겟, 최대 200자)
6. 체크리스트 10개 항목 검증
7. AI 말투 최종 점검 — 하나라도 있으면 수정
8. 사용자에게 전달 (시훈이 검토 후 게시)
```

**참조 파일:**
- `src/agents/content.md` — 빈이 페르소나, 6단계 CoT, 워밍업 모드
- `src/agents/post-writing-guide.md` — 이 문서 (조회수 패턴 + 글쓰기 공식)
- `src/agents/youtube-needs-analyzer.md` — 니즈 분석 → 소재 발굴
