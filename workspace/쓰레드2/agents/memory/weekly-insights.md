# 주간 인사이트 (Weekly Insights)

> 서연(분석가)이 매주 일요일 작성. TOP/BOTTOM 분석 + 다음 주 전략.

---

## 사용 방법

매주 일요일 새 항목 추가 (append-only).

```
## YYYY-WW (YYYY년 N주차)
기간: YYYY-MM-DD ~ YYYY-MM-DD

### 성과 요약
- 총 포스트: N개
- 평균 조회수: N뷰
- 최고 조회수: N뷰 ([post_id])
- 총 좋아요: N개

### TOP 3 포스트
1. [post_id] — N뷰 — 패턴: [A~E] — 카테고리: [뷰티/건강/생활/다이어트]
2. [post_id] — N뷰
3. [post_id] — N뷰

### BOTTOM 3 포스트
1. [post_id] — N뷰 — 원인 분석: [...]
2. [post_id] — N뷰
3. [post_id] — N뷰

### 패턴 분석
- 잘 된 포맷: [포맷명] (N개 중 N개 성공)
- 안 된 포맷: [포맷명]
- 잘 된 훅 유형: [질문형/반전형/...]
- 잘 된 카테고리: [카테고리]

### 다양성 체크
- 포맷 분포: 문제공감형 N% / 솔직후기형 N% / ...
- 카테고리 분포: 뷰티 N% / 건강 N% / ...
- 경고: [편중 있으면 명시]

### 다음 주 권고
- 강화할 것: [...]
- 줄일 것: [...]
- 실험 제안: [...]
```
