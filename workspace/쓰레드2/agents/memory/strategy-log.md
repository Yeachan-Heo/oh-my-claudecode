# 전략 로그 (Strategy Log)

---

## 사용 방법

새 항목은 파일 끝에 추가만 한다. 수정/삭제 금지.

```
## YYYY-MM-DD

### 결정
- 카테고리 비율: 뷰티 4 / 건강 3 / 생활 2 / 다이어트 1
- 실험 슬롯: 포스트 8~10번 (시간대 실험)
- 리사이클 후보: [post_id]

### 결과 (다음날 기록)
- 총 조회수: N뷰
- 최고 포스트: [post_id] N뷰
- 실험 결과: [가설] → [검증 결과]

### 메모
- 특이사항, 다음날 반영사항
```
