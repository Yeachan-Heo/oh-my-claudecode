## 2. 분석 파이프라인 결과

### 추출된 니즈 (4개)

| 카테고리 | 문제 | 신호강도 | 구매연결 | Threads 적합도 | 포스트 수 |
|---------|------|---------|---------|---------------|----------|

| 불편해소 | 육아 피로/스트레스 | L1 | 하 | 3 | 5 |

### 니즈 품질 평가
- **"오후 집중력 저하"**: 구매 연결 높음 (카페인 음료, 영양제 매칭 가능). 실제 제품 매칭 가능.
- **"아이 아토피 관리"**: 구매 연결 중간. 보습제/크림 매칭 가능하나 민감 카테고리(의료).
- **"제휴마케팅 시작"**: 마케터 니즈로 소비자 니즈 아님. 구매 연결 낮음.
- **"육아 피로"**: 공감형 콘텐츠 소재로 좋으나 직접 구매 연결 약함.

### 매칭된 상품: 8개
- 니즈 4개에 대해 기존 상품 DB(50개)에서 8개 매칭 완료.

## 3. 현재 수집 전략의 문제점

### 3-1. 키워드가 마케터/판매자 중심
현재 키워드("쿠팡파트너스", "제휴마케팅")로 발굴되는 채널은 **이미 제휴마케팅을 하는 사람들**이다.
- 이들의 포스트는 대부분 광고성(쓰레드형 + 쿠팡링크)
- 소비자의 진짜 니즈(질문, 고민, 추천 요청)가 거의 없음
- 57개 중 질문/니즈 패턴 매칭: 8개(14%)에 불과

### 3-2. 채널 다양성 부족
- 수집된 6개 채널 모두 "마케터" 유형
- 일반 소비자 채널 0개
- 카테고리 편중: 경제/투자(itsmyturn85), 뷰티(orion_on82), 크리에이터(yeongyeong_studio)

### 3-3. 조회수 양극화
- 대형 채널(@itsmyturn85, @orina_blog)의 조회수가 압도적(15만, 11만)
- 소형 채널(@joon9_77)은 평균 268회로 노출 미미
- 팔로워 1,000~5,000 구간이 가장 효율적 (engagement rate 높음)

## 4. 수집 전략 제안

### 4-1. 효과적인 키워드 목록

**A. 소비자 니즈 발견 키워드 (최우선)**
| 키워드 | 목적 | 예상 발견 유형 |
|--------|------|--------------|
| 추천해줘 | 직접 구매 의향 | 구매 고민 포스트 |
| 뭐가좋아 | 비교 검색 | 카테고리 탐색 |
| 써본사람 | 후기 탐색 | 제품 리뷰 |
| 살까말까 | 구매 결정 단계 | 고민 공유 |

| 이거써봤는데 | 사용 경험 공유 | 솔직 후기 |

**B. 카테고리별 니즈 키워드**
| 카테고리 | 키워드 |
|---------|--------|

| 육아 | 육아템, 이유식추천, 기저귀추천, 분유비교 |
| 생활 | 자취꿀팁, 청소팁, 정리정돈, 집꾸미기 |
| 식품 | 간편식추천, 밀키트, 다이어트식단, 건강간식 |
| 디지털 | 이어폰추천, 태블릿, 노트북추천, 가성비폰 |

**C. 감성/공감 키워드**
| 키워드 | 목적 |
|--------|------|
| 인생템 | 강한 구매 만족 신호 |
| 꿀템 | 가성비 좋은 제품 발견 |
| 갓성비 | 가격 대비 만족도 |
| 이건사야해 | 강한 구매 추천 |
| 후회없는 | 구매 확신 |

### 4-2. 피해야 할 키워드/채널 유형

**피해야 할 키워드:**
- 쿠팡파트너스, 제휴마케팅, 파트너스수익, 링크수익 (마케터만 잡힘)
- 할인코드, 공구, 타임세일 (광고성 채널만 잡힘)

**피해야 할 채널 유형:**
- 팔로워 50 미만: 활동량 너무 적어 데이터 가치 낮음
- 팔로워 50,000 이상: 연예인/인플루언서급, 제휴마케팅 부적합
- 광고 비율 80% 이상: 순수 광고 채널, 소비자 니즈 없음
- 봇/자동화 채널: 포스트 패턴 반복, 대화 없음
- 비한국어 채널: 타겟 시장 불일치

### 4-3. 채널 필터링 기준 개선안

**현재 기준:**
- 팔로워 >= 100 AND 광고 포스트 >= 1 AND 최근 활동

**개선안:**
```
소비자 니즈 채널:
  - 팔로워 >= 100
  - 최근 7일 내 포스트 >= 1
  - 포스트 중 질문/고민/추천요청 패턴 >= 1
  - 광고 비율 < 50% (순수 소비자 목소리 확보)

마케터 채널 (기존):
  - 팔로워 >= 100
  - 광고 포스트 >= 1
  - 최근 7일 내 활동
```
