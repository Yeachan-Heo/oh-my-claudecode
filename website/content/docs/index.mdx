## Oh My ClaudeCode란?

Oh My ClaudeCode(OMC)는 Claude Code 위에서 돌아가는 멀티 에이전트 오케스트레이션 레이어입니다.

프롬프트 하나로 여러 전문 에이전트를 조율해서 복잡한 개발 작업을 자동화합니다.

## 주요 기능

- 20개 전문 에이전트 — 탐색, 계획, 구현, 검증, 리뷰까지 역할 분담
- 28개 스킬 — autopilot, ralph, ultrawork 등 자동화 워크플로우
- 매직 키워드 — "autopilot build me X" 한 마디로 전체 파이프라인 실행
- 팀 오케스트레이션 — 여러 Claude 에이전트를 동시에 조율
- 상태 관리 — notepad, project-memory로 컨텍스트 유지

## 시작하기
