## 개요

Vision은 이미지, 스크린샷, 다이어그램 등 시각적 자료를 분석하는 에이전트입니다.
Claude의 멀티모달 기능으로 이미지 내용을 파악하고 분석 결과를 돌려줍니다.

UI 스크린샷에서 레이아웃 문제를 발견하거나,
아키텍처 다이어그램을 해석하거나,
에러 스크린샷의 내용을 파악하는 데 씁니다.

읽기 전용 에이전트로, 파일을 수정하지 않습니다.

## 언제 쓰는가

- UI 스크린샷의 레이아웃이나 디자인 문제를 분석할 때
- 아키텍처 다이어그램이나 플로우차트를 해석할 때
- 에러 스크린샷의 내용을 파악할 때
- 시각적 결과물의 품질을 검토할 때

## 사용 예시

```
"이 스크린샷에서 UI 문제를 찾아줘"
"이 아키텍처 다이어그램을 분석해줘"
"이 에러 화면의 내용을 파악해줘"
```

## 분석 대상

- **UI 스크린샷**: 레이아웃, 정렬, 반응형, 접근성 문제
- **다이어그램**: 아키텍처, 시퀀스, ERD, 플로우차트 해석
- **에러 화면**: 에러 메시지, 스택 트레이스, 콘솔 출력
- **디자인 목업**: 디자인 의도 파악과 구현 가이드

## 다른 에이전트와의 조합

- **designer**: Vision이 UI 문제를 발견하면 Designer가 수정을 구현합니다
- **architect**: 다이어그램 분석 결과를 바탕으로 Architect가 아키텍처를 검토합니다
- **debugger**: 에러 스크린샷 분석 결과를 바탕으로 Debugger가 근본 원인을 추적합니다
