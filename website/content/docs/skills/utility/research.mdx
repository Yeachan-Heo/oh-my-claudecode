## 개요

Research는 [SciOMC](/docs/skills/workflow/sciomc)의 연구 기능을 호출하는 유틸리티 진입점입니다. 복잡한 주제를 독립 조사 단계로 분해하고 병렬 scientist 에이전트를 실행합니다.

## 사용법

```bash
/oh-my-claudecode:sciomc <research goal>
/oh-my-claudecode:sciomc AUTO: <research goal>
```

자세한 사용법은 [SciOMC](/docs/skills/workflow/sciomc) 페이지를 참고.

## 관련 스킬

- [sciomc](/docs/skills/workflow/sciomc) - 전체 연구 오케스트레이션 문서
- [external-context](/docs/skills/workflow/external-context) - 외부 문서 조회
- [trace](/docs/skills/utility/trace) - 증거 기반 인과 추적
