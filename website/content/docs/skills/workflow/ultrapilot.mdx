## 개요

Ultrapilot은 여러 autopilot 워커를 파일 소유권 기반으로 병렬 실행하던 스킬이었습니다.

**v4.1.7부터 deprecated되어 [team](/docs/skills/workflow/team) 모드로 대체됐다.**

`ultrapilot`을 호출하면 자동으로 team 모드로 라우팅됩니다.

## 대체 방법

기존 ultrapilot 사용 패턴은 team으로 바꾸면 됩니다.

```bash
# 이전 (deprecated)
/oh-my-claudecode:ultrapilot "build a fullstack todo app"

# 현재 (권장)
/oh-my-claudecode:team 3:executor "build a fullstack todo app"
```

## team과의 비교

| 항목 | Ultrapilot (deprecated) | Team (권장) |
|------|------------------------|-------------|
| 워커 관리 | 파일 소유권 기반 | 작업 목록 기반 |
| 조율 방식 | 상태 파일 | 네이티브 팀 메시징 |
| 파이프라인 | 없음 | 5단계 (plan → prd → exec → verify → fix) |
| Ralph 통합 | 없음 | `team ralph` 지원 |

## 관련 스킬

- [team](/docs/skills/workflow/team) - 권장 대체 스킬
- [autopilot](/docs/skills/workflow/autopilot) - 단일 자율 실행 파이프라인
